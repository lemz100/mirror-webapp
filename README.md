# react-fullstack

## *REACT FULLSTACK TEMPLATE*

Has:
- ESLint
- Prettier
- Webpack
- React
- Node.js w/ Express.js
- A route out the box

commands:
client/: npm start = Runs React app

client/: npm run build = Webpack builds dist for deploying

client/: npx eslint . = Runs eslint (not needed, it does this automatically)

server/: node server
